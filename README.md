# 🛒 E-Commerce Sales Dashboard

An interactive **Power BI dashboard** created to analyze e-commerce sales data and understand business performance through clear visualizations. This project helped me practice data cleaning, data modeling, DAX, and dashboard design while working with a real-world dataset.

---

# 📌 Project Overview

This dashboard provides an overview of an e-commerce business by tracking important metrics such as total sales, profit, quantity sold, and average order value (AOV). It also helps analyze customer purchasing behavior, product categories, payment methods, and sales across different states.

The goal of this project is to convert raw sales data into meaningful insights that can help businesses understand their performance and make better decisions.

---

# 🎯 Project Objective

The main objective of this dashboard is to:

* Monitor overall sales performance.
* Track profit and revenue.
* Analyze customer purchasing patterns.
* Compare sales across different states.
* Identify the most profitable product categories.
* Understand customer payment preferences.
* Present business data in an easy-to-understand visual format.

---

# 🛠️ Technologies Used

* **Power BI Desktop** – Created the dashboard and reports.
* **Power Query** – Cleaned and transformed the data.
* **DAX (Data Analysis Expressions)** – Created calculated measures such as Average Order Value (AOV) and KPIs.
* **Data Modeling** – Built relationships between the Orders and Details tables.
* **PBIX File** – Power BI project file.
* **PNG Images** – Dashboard screenshots.

---

# 📂 Dataset

**Source:** Kaggle (used through a YouTube tutorial)

The dataset contains information about e-commerce orders and is divided into two related tables.

### Orders Table

* Order ID
* Order Date
* Customer Name
* State

### Details Table

* Category
* Sub-Category
* Quantity
* Sales Amount
* Profit
* Payment Mode

Both tables are connected using **Order ID**, allowing Power BI to combine the data for analysis.

---

# 📊 Dashboard Features

### 📌 KPI Cards

Displays important business metrics at a glance:

* Total Sales Amount
* Total Profit
* Total Quantity Sold
* Average Order Value (AOV)

### 📅 Profit by Month

Shows monthly profit trends to identify months with higher or lower profitability.

### 📦 Profit by Sub-Category

Compares the profit earned from different product sub-categories and highlights the most profitable products.

### 🛍️ Quantity by Category

Displays the number of products sold in categories such as Furniture, Electronics, and Clothing.

### 💳 Quantity by Payment Mode

Shows how customers prefer to pay for their orders, including:

* Cash on Delivery (COD)
* UPI
* Debit Card
* Credit Card
* EMI

### 🗺️ Sales by State

Compares total sales across different states to identify the best-performing regions.

### 👤 Sales by Customer

Highlights customers who contributed the highest sales amount.

### 🎛️ Interactive Slicers

Users can filter the dashboard by:

* Quarter
* State

This makes it easy to explore specific regions or time periods.
